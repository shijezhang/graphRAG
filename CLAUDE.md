# GraphRAG V2

## Superpowers Skills

This project uses [superpowers](https://github.com/obra/superpowers) skills for structured development workflows.

Available skills (invoke via `/command-name`):
- `/brainstorming` — Design refinement before implementation
- `/writing-plans` — Create implementation plans
- `/executing-plans` — Execute plans systematically
- `/test-driven-development` — TDD workflow
- `/systematic-debugging` — Structured debugging approach
- `/dispatching-parallel-agents` — Coordinate parallel subagents
- `/subagent-driven-development` — Drive development with subagents
- `/using-git-worktrees` — Isolated workspaces for tasks
- `/requesting-code-review` — Request code review
- `/receiving-code-review` — Process code review feedback
- `/finishing-a-development-branch` — Complete and merge branches
- `/verification-before-completion` — Verify work before declaring done
- `/writing-skills` — Create new skills

When starting creative or implementation work, invoke `/brainstorming` first to refine the design before writing code.
